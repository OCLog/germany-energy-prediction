# Literature Review

Approaches or solutions that have been tried before on similar projects.

**Summary of Each Work**:

- **Source 1**: Electrical Load Forecasting: A methodological overview

  - **[Link](https://www.researchgate.net/publication/349451251_Electrical_Load_Forecasting_A_methodological_overview)**
  - **Objective**: Methodological overview for a large energy load dataset; Finding the best performing models
  - **Methods**: Conventional, hybrid and AI models
  - **Outcomes**: AI models are most popular and RNNs / LSTMs suggest promising results
  - **Relation to the Project**: Same data domain

- **Source 2**: Ultra-short-term Load Forecasting Based on XGBoost-BiGRU

  - **[Link](https://www.researchgate.net/publication/383683988_Ultra-short-term_Load_Forecasting_Based_on_XGBoost-BiGRU)**
  - **Objective**: Chinese Load data forecasting
  - **Methods**: XGBoost and bidirectional gated recurrent unit (BiGRU)
  - **Outcomes**: Combining XGBoost and NN works better than single models
  - **Relation to the Project**: Same data domain
